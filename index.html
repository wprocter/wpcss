Responsive Display Utilities

Show or hide elements responsively using .d-none, .d-block, .d-sm-block, .d-md-none, etc. These work like Bootstrapâ€™s display utilities. Combine classes for precise control:

  • .d-none: Hidden on all breakpoints
  • .d-block: Visible on all breakpoints
  • .d-none .d-sm-block: Hidden only on xs, visible on sm and up
  • .d-sm-none .d-md-block: Hidden only on sm, visible on md and up
  • .d-none .d-md-block .d-lg-none: Visible only on md

Buttons

Recommended: Use .btn with .bg-* and .t-* utility classes for flexible, accessible button colors.
Alternative: Use predefined color button classes like .btn-blue-4 for convenience.

<!-- Utility-first approach --> <button class="btn bg-blue-4 t-white">Utility Blue</button> <button class="btn bg-green-5 t-white">Utility Green</button> <button class="btn bg-red-5 t-white">Utility Red</button> <!-- Predefined color classes (for convenience) --> <button class="btn btn-blue-4">.btn-blue-4</button> <button class="btn btn-green-5">.btn-green-5</button> <button class="btn btn-red-5">.btn-red-5</button>
